# Test Fixtures

This directory contains test data files for the PDF to DICOM converter.

## Contents

- `radiology_report.pdf` - Sample radiology report PDF file for testing conversion
- `corrupted.pdf` - Corrupted PDF file for error handling tests
- `metadata/` - Sample metadata JSON files

## Usage

These files are used by the test suite to validate PDF to DICOM conversion functionality. The `radiology_report.pdf` contains a sample medical report with charts and imaging data suitable for testing DICOM encapsulation.
